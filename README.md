#  Customer Profile Analysis

## Project Overview
Customer Profile Analysis is a data analytics project focused on understanding customer demographics, socio-economic conditions, geographic distribution, and behavioral patterns using data wrangling and visualization techniques.

The project converts raw customer data into meaningful insights through an end-to-end data processing pipeline including data cleaning, transformation, analysis, and dashboard visualization.

The dataset contains customer attributes such as:

- Age
- Gender
- Education Level
- Salary
- Family Size
- Subscription Type
- Account Status
- Purchase Activity
- City Location

The goal of the project is to identify customer patterns and support data-driven decision-making.

---

## Objectives
- Analyze customer demographic distribution
- Study purchasing behavior trends
- Understand subscription engagement
- Identify geographic performance differences
- Generate business insights from data

## Results & Insights
- Married customers form a major customer segment.
- Educated customers show higher engagement.
- Purchase activity remains consistent across years.
- Certain cities show higher economic value.
- Older customers tend to have higher salary levels.
